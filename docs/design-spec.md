# 设计规范

## 色彩系统

### 主色板 - 淡紫系列
| 色值 | 用途 |
|------|------|
| `#C9B1FF` | 主按钮、强调色 |
| `#E8DCF0` | 卡片背景、次要区域 |
| `#F5F0F8` | 页面背景 |
| `#A88FD4` | 文字链接、图标 |
| `#8B6FC0` | 深色文字、hover 状态 |

| 色值 | 用途 |

## 组件风格指南

### 博主卡片（便签风格）
- 背景带轻微倾斜（rotate: -1deg 到 1deg 随机）
- 顶部有胶带/图钉装饰
- 卡片颜色从柔和色板中随机选取
- Hover 时卡片微微上浮

### 按钮
- 主按钮：淡紫色背景 + 白色文字 + 圆角胶囊形
- 次要按钮：淡紫边框 + 透明背景
- 开关：自定义滑动开关，紫色激活态
- 危险操作（删除）：浅红色，确认后执行

### 装饰元素
- 页面角落点缀小花朵 SVG
- 卡片上随机散布小星星
- 加载动画：跳动的紫色圆点
- 空状态：插画 + 引导文字

## 动效

- 卡片 hover：translateY(-4px) + shadow 增大，过渡 0.3s ease
- 开关切换：背景色渐变，过渡 0.2s ease
- 弹窗进出：scale(0.9→1) + opacity(0→1)，过渡 0.25s ease-out
- 列表新增/删除：fade + slide，过渡 0.3s ease
