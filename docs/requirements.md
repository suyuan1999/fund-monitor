# 产品需求文档

## 项目背景

用户在 Mac 上追踪小红书基金博主，获取每日基金操作建议，辅助加减仓决策。

## 核心需求

### 1. 博主管理
- 添加博主：粘贴小红书链接，自动解析昵称等信息
- 博主列表：卡片式展示，包含昵称、关注日期、最新更新时间
- 搜索博主：按昵称实时检索
- 删除博主：确认后级联删除关联数据
- 编辑博主：修改昵称、标签备注

### 2. 内容获取
- 一键拉取：点击按钮自动解析博主最新笔记
- 批量拉取：一键更新全部博主最新内容
- 手动补录：解析失败时弹出文本框手动粘贴
- 拉取状态反馈：成功/失败/未拉取 三种状态标识

### 3. 分析看板
- 今日汇总：各博主今日操作（加仓/减仓/观望 + 具体基金）
- AI 综合分析：综合所有博主观点的加减仓建议
- 历史回溯：日历视图，点击日期查看历史分析

### 4. 费用控制
- 首页顶部 AI 分析开关（关闭时分析按钮置灰）
- 首页顶部推送开关（关闭时推送按钮置灰）
- 所有花钱操作手动触发，双重保险
- 分析按钮旁显示预估费用

### 5. 推送功能
- 手动推送到微信（通过 PushPlus）
- 每日定时本地提醒（不花钱）
- 推送内容：博主观点摘要 + AI 综合建议

### 6. 设置中心
- PushPlus Token 配置
- AI API Key 配置
- 每日提醒时间
- 小红书 Cookie（可选）

## 非功能需求
- 桌面应用，Mac 平台运行
- 轻量级，启动迅速
- UI 简洁文艺童趣，淡紫色主题
- 数据本地存储（SQLite），无需网络即可查看历史
- 可打包为 .dmg 安装包
